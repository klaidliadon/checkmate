# Checkmate
Find all unique configurations of a custom chess board where none of the pieces is in a position to take any of the others.

## Usage

	java -jar checkmate.jar <config.properties>
	
## Configuration

	# board size
	board.width = 7
	board.height = 7
	# number of pieces
	queen = 2
	rook = 0
	bishop = 2
	knight = 1
	king = 2
